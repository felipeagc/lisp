Lisp

Lisp é uma família de linguagens de programação concebida por John McCarthy em 1958.

Sintaxe

(funcao param1 param2 ...)

Exemplo: operações básicas

(+ 1 2 3) => 6.0
(- 3 1) => 2.0
(* 2 3) => 6.0
(/ 15 3) => 5.0

Exemplo: variaveis

(set x 2)

x => 2.0

(* x 3) => 6.0

Exemplo: if

(if cond vardadeiro falso)

Exemplo: if

(if (< 1 2)
  (+ 1 2)
  (- 1 2)) => 3.0

Exemplo: if

(if (not (< 1 2))
  (+ 1 2)
  (- 1 2)) => -1.0

Exemplo: definição de funções

(set add1 (fun (n) (+ 1 n)))

(add1 4) => 5.0

Exemplo: definição de funções

(defun add1 (n) (+ 1 n))

(add1 4) => 5.0

Exemplo: fatorial

(defun fat (n)
  (if (= n 1)
    1
    (* n (fat (- n 1)))))

(fat 5) => 120.0

Exemplo: sequência de operações

(progn
  (set a 1.0)
  (set b 2.0)
  (+ a b)) => 3.0
